Is continuous CoT better suited for multi-lingual reasoning?

  • 作者：Ali Hamza Bashir, Behzad Shomali, Markus Frey, Mehdi Ali, Rafet Sifa, David Berghaus
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：9.47/10

一句话总结：该工作表明，把推理过程放入连续潜空间，可能比显式文字 CoT 更适合多语言泛化，尤其对低资源语言和 zero-shot 场景更有效。

核心贡献/观点： - 系统比较连续 Chain-of-Thought 与标准监督微调在多语言推理中的表现。 - 覆盖英语、中文、德语、法语、乌尔都语五种语言，验证语言不变性的实际收益。 - 在低资源语言与未见语言 zero-shot 设置下，连续推理明显优于显式推理。 - 推理轨迹还可压缩约 29x-50x，兼顾泛化与效率。

关键结果：在 GSM8k 和 CommonsenseQA 上，continuous reasoning 在低资源语言上的鲁棒性更强，并显著压缩推理痕迹长度。

DyLLM: Efficient Diffusion LLM Inference via Saliency-based Token Selection and Partial Attention

  • 作者：Younjoo Lee, Junghoo Lee, Seungkyun Dan, Jaiyoung Park, Jung Ho Ahn
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：8.80/10

一句话总结：DyLLM 利用相邻扩散步之间 token 表示的稳定性，只重算真正“显著”的 token，大幅提升扩散式 LLM 解码吞吐。

核心贡献/观点： - 观察到扩散式 LLM 在迭代去噪过程中，多数 token 的表示变化很小。 - 提出基于 attention context 余弦相似度的 saliency 判定机制，找出需要更新的 token。 - 仅对 salient tokens 重新执行 FFN 和 attention，其余 token 复用缓存激活。 - 无需重新训练即可部署，是一种 training-free 的 inference acceleration 框架。

关键结果：在 reasoning 和 code generation 基准上，DyLLM 可达到最高 9.6x 吞吐提升，同时大体保持 LLaDA、Dream 等基线模型精度。

FedMomentum: Preserving LoRA Training Momentum in Federated Fine-Tuning

  • 作者：Peishen Yan, Yang Hua, Hao Wang, Jiaru Zhang, Xiaoyu Wu, Tao Song, Haibing Guan
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：8.80/10

一句话总结：FedMomentum 通过对正确聚合后的低秩更新做 SVD 重构，尽量保留联邦 LoRA 微调中的跨轮次训练动量。

核心贡献/观点： - 指出直接分别平均 LoRA 的 down/up 矩阵在数学上并不正确，会引入额外噪声。 - 认为现有无噪声聚合方法又会牺牲 LoRA 的结构表达能力与客户端个性化信息。 - 将问题抽象为“训练动量丢失”，即多轮训练更新无法有效累积。 - 提出基于 SVD 的 momentum-preserving 聚合方式，并支持保留 residual 信息增强鲁棒性。

关键结果：在多个任务上，FedMomentum 相比现有方法在收敛速度和最终精度上都更稳定、更优。

Foley-Flow: Coordinated Video-to-Audio Generation with Masked Audio-Visual Alignment and Dynamic Conditional Flows

  • 作者：Shentong Mo, Yibing Song
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：8.53/10

一句话总结：Foley-Flow 通过 masked audio-visual alignment 与动态条件 flow，同步提升视频到音频生成的语义一致性和节奏对齐。

核心贡献/观点： - 指出以往对比学习式 AV 对齐更擅长语义匹配，但对时序节奏同步支持不足。 - 使用 masked modeling 训练方式，让视频片段直接指导音频片段恢复，增强局部对应关系。 - 在生成阶段引入 dynamic conditional flow，用时变视频特征动态调制音频生成。 - 同时建模 semantic consistency 与 rhythmic consistency，更适合真实 Foley 生成场景。

关键结果：在标准 benchmark 上，Foley-Flow 在多项指标上明显优于现有方法，音频与视频的语义和节奏协调性更强。

CDRRM: Contrast-Driven Rubric Generation for Reliable and Interpretable Reward Modeling

  • 作者：Dengcan Liu, Fengkai Yang, Xiaohan Wang, Shurui Yan, Jiajun Chai, Jiahao Li, Yikun Ban, Zhendong Mao, Wei Lin, Guojun Yin
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：8.13/10

一句话总结：CDRRM 先做对比式偏好剖析，再自动综合出高质量 rubric，以更可解释、更高数据效率的方式完成 reward modeling。

核心贡献/观点： - 传统 reward model 依赖昂贵标注且可解释性差，rubric-based 方法又常产生冗余和噪声标准。 - 提出 Contrast-then-Synthesis 范式，先识别偏好样本中的关键因果差异，再生成紧凑 rubric。 - 这些 rubric 可直接指导 preference judgment，并减少 verbosity / position 等评估偏差。 - 通过少量高质量样本训练 rubric generator，就能增强冻结 judge model 的表现。

关键结果：在 RewardBench、RMBench、RMB 上取得 SOTA，并且仅用 3k 高质量样本就能让冻结判别模型超过完全微调基线。

Reject, Resample, Repeat: Understanding Parallel Reasoning in Language Model Inference

  • 作者：Noah Golowich, Fan Chen, Dhruv Rohatgi, Raghav Singhal, Carles Domingo-Enrich, Dylan J. Foster, Akshay Krishnamurthy
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：8.13/10

一句话总结：该论文用 particle filtering / SMC 理论重新理解并行 reasoning 推理策略，为 accuracy-cost tradeoff 提供了更严格的分析框架。

核心贡献/观点： - 把 inference-time 采样、聚合、裁剪等并行 reasoning 方法统一到 Sequential Monte Carlo 视角下。 - 研究在给定 process reward evaluations 预算时，如何更准确地逼近目标分布。 - 从理论上给出非渐近保证成立的简单条件，并讨论 SMC 的算法改进方向。 - 指出所有 particle filtering 方法都面临某种基本极限，说明并行 reasoning 并非“免费午餐”。

关键结果：实证结果显示，理论条件确实能较好解释 SMC 的 sampling error，但最终 accuracy 仍受更多因素影响。

Dissecting Quantization Error: A Concentration-Alignment Perspective

SLO-Aware Compute Resource Allocation for Prefill-Decode Disaggregated LLM Inference

  • 作者：Luchang Li, Dongfang Li, Bozhao Gong, Yu Zhang
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 详细报告：20_Research/Papers/大模型/SLO-Aware_Resource_Allocation

MoEless: Efficient MoE LLM Serving via Serverless Computing

  • 作者：Hanfei Yu, Bei Ouyang, Shwai He, Ang Li, Hao Wang
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 详细报告：20_Research/Papers/大模型/2026-03/MoEless_Serverless-MoE-Serving
  • 评分：11.5/10

一句话总结：moeless 把无服务器专家扩展引入 moe 推理服务，在真实工作负载上同时显著降低延迟与成本。

核心贡献/观点： - 针对 moe 专家负载不均衡导致的 straggler 问题提出 serverless serving 框架 - 用轻量 layer-aware predictor 预测专家负载并提前识别热点 - 通过 expert scaling + placement 提升 locality、GPU 利用率与负载均衡

NOBLE: Accelerating Transformers with Nonlinear Low-Rank Branches

  • 作者：Ethan Smith
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：10.3/10

Contrastive-to-Self-Supervised: A Two-Stage Framework for Script Similarity Learning

  • 作者：Claire Roman, Philippe Meyer
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：9.5/10

一句话总结：该工作提出两阶段框架，在缺乏可靠历史标签时学习跨文字系统的相似性表示。

核心贡献/观点： - 先在 invented alphabets 上做 supervised contrastive learning 获得 teacher encoder - 再通过 teacher-student distillation 将知识迁移到真实历史文字系统 - 兼顾 hard boundaries 与 soft similarities，适合争议性历史关系建模

关键结果：在 few-shot glyph recognition 和 script clustering 上取得有效结果。

FlashPrefill: Instantaneous Pattern Discovery and Thresholding for Ultra-Fast Long-Context Prefilling

TaPD: Temporal-adaptive Progressive Distillation for Observation-Adaptive Trajectory Forecasting in Autonomous Driving

核心贡献/观点： - 提出 Observation-Adaptive Forecaster 处理不同历史长度输入 - 用 progressive knowledge distillation 把长历史教师知识迁移给短历史学生 - 再通过 Temporal Backfilling Module 重建缺失历史以增强极短观测场景

关键结果：在 Argoverse 1/2 上对不同 observation lengths 都优于强基线。

Stem: Rethinking Causal Information Flow in Sparse Attention

  • 作者：Lin Niu, Xin Luo, Linchuan Xie, Yifu Sun, Guanghua Yu, Jianchen Zhu, S Kevin Zhou
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 评分：8.5/10

关键结果：实验显示在降低计算与 prefill latency 的同时取得更高精度。
