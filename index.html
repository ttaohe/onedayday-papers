Balancing Coverage and Draft Latency in Vocabulary Trimming for Faster Speculative Decoding

  • 作者：Ofir Ben Shoham
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 详细报告：20_Research/Papers/大模型/Vocabulary-Trimming_Speculative-Decoding
  • 笔记：已有详细分析

一句话总结：通过词汇裁剪优化投机解码中的草稿模型，在保持覆盖率的同时降低延迟，实现高达 20% 的吞吐量提升。

核心贡献/观点： - 投机解码的瓶颈在于草稿模型的语言建模头随词汇量增长而变慢 - 提出将草稿词汇选择建模为约束优化问题，平衡覆盖率和延迟 - 使用 Tree-structured Parzen Estimator 优化覆盖率-延迟帕累托前沿 - 草稿词汇可减少高达 97%，同时保持高覆盖率

关键结果：在领域特定任务上实现 16% 延迟降低和 20% 吞吐量提升，在分布外任务上也获得 6.7% 吞吐量增益。

POET-X: Memory-efficient LLM Training by Scaling Orthogonal Transformation

  • 作者：Zeju Qiu, Lixin Liu, Adrian Weller, Han Shi, Weiyang Liu
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 详细报告：20_Research/Papers/大模型/POET-X_Memory-efficient-LLM-Training
  • 笔记：已有详细分析

一句话总结：提出 poet-x，一种可扩展且内存高效的大模型训练变体，通过正交等价变换在单张 H100 GPU 上预训练十亿参数级别的 llm。

核心贡献/观点： - poet-x 是 POET 框架的改进版本，显著降低了内存消耗和计算开销 - 通过减少矩阵乘法计算成本实现高效的训练吞吐量 - 在单个 Nvidia H100 GPU 上实现十亿参数 llm 的预训练 - 在相同设置下，AdamW 等标准优化器会耗尽显存

关键结果：poet-x 在保持 POET 的泛化和稳定性优势的同时，实现了显著的吞吐量和内存效率提升。

Mixture of Universal Experts: Scaling Virtual Width via Depth-Width Transformation

  • 作者：Yilong Chen, Naibin Gu, Junyuan Shang, Zhenyu Zhang, Yuchen Feng, Jiawei Sheng, Tingwen Liu, Shuohuan Wang, Yu Sun, Hua Wu, Haifeng Wang
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 详细报告：20_Research/Papers/大模型/MOUE_Virtual-Width-MoE
  • 笔记：已有详细分析

一句话总结：提出 MOUE (Mixture of Universal Experts)，通过深度-宽度转换引入虚拟宽度新维度，打破 moe 模型的物理尺寸限制。

核心贡献/观点： - MOUE 引入虚拟宽度作为新的缩放维度，重用跨层的通用专家池 - 设计交错旋转拓扑实现结构化专家共享 - 提出通用专家负载平衡进行深度感知的曝光修正 - 通用路由器使用轻量级轨迹状态实现连贯的多步路由

Dissecting Quantization Error: A Concentration-Alignment Perspective

  • 作者：Marco Federici, Boris van Breugel, Paul Whatmough, Markus Nagel
  • 链接：arXiv | PDF
  • 来源：[arXiv]
  • 详细报告：20_Research/Papers/大模型/Quantization-Error_Concentration-Alignment
  • 笔记：已有详细分析

一句话总结：首次将 SQNR 分解为 Concentration 和 Alignment 两个独立分量，提出 CAT 变换联合优化两者，在 4-bit 量化下超越现有方法。

核心贡献/观点： - 从理论上将线性层量化的 SQNR 分解为集中度和对齐度 - 揭示 Alignment 是被现有量化方法忽略的关键因素 - 提出 CAT (Concentration-Alignment Transform) 联合优化两个分量 - 使用小规模校准集估计协方差，实现后训练量化

关键结果：在多个 LLM 上，CAT 在 4-bit 精度下始终匹配或超越现有变换量化方法，SQNR 提升 4.1%，PPL 降低 8.4%。

Marco Federici, Boris van Breugel, Paul Whatmough, Markus Nagel
arXivPDF
首次将 SQNR 分解为 Concentration 和 Alignment 两个独立分量，提出 CAT 变换联合优化两者，在 4-bit 量化下超越现有方法。
核心贡献
  • 从理论上将线性层量化的 SQNR 分解为集中度和对齐度
  • 揭示 Alignment 是被现有量化方法忽略的关键因素
  • 提出 CAT (Concentration-Alignment Transform) 联合优化两个分量
  • 使用小规模校准集估计协方差，实现后训练量化
关键结果

在多个 LLM 上，CAT 在 4-bit 精度下始终匹配或超越现有变换量化方法，SQNR 提升 4.1%，PPL 降低 8.4%。

SLO-Aware Compute Resource Allocation for Prefill-Decode Disaggregated LLM Inference

Luchang Li, Dongfang Li, Bozhao Gong, Yu Zhang
arXivPDF
提出首个 Prefill-Decode 解耦系统的资源分配理论，结合 M/M/1 排队论与经验基准测试，在满足 SLO 约束下实现 15-30% 的资源节省。

  • 结合理论建模与经验测量的混合方法论

在所有测试场景下预测的资源分配与实际最优值完全一致，相比静态分配方法节省 15-30% 的 GPU 资源。
