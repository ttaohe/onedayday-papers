BTZSC: A Benchmark for Zero-Shot Text Classification Across Cross-Encoders, Embedding Models, Rerankers and LLMs

Ilias Aarab 8.7/10 arXiv ↗ PDF ↗
📄 摘要
零样本文本分类希望通过将文本直接与人类可读的标签描述进行匹配，从而摆脱昂贵的任务标注成本。早期方法主要依赖基于自然语言推理微调的 cross-encoder，但近期 embedding model、reranker 和 instruction-tuned LLM 的进展正在挑战这一范式。作者提出 BTZSC，一个包含 22 个公开数据集的综合 benchmark，覆盖情感、主题、意图与情绪分类等任务，并系统比较四类模型家族共 38 个 checkpoint。结果显示：reranker 达到新的 SOTA；强 embedding model 在精度与延迟之间给出最佳 trade-off；4B 到 12B instruction-tuned LLM 已具备较强竞争力；传统 NLI cross-encoder 的扩展收益有限。
🔍 研究问题与背景
  • 研究背景：zero-shot 分类的真实部署场景越来越关注延迟、吞吐和无需微调的泛化能力。
  • 研究动机：现有评测往往混入 supervised probe 或 fine-tuning，无法真实刻画纯 zero-shot 能力。
  • 核心方法：构建统一 benchmark，对四类模型在 22 个数据集上做系统比较。
  • 主要结果：Qwen3-Reranker-8B 取得 macro F1 0.72；GTE-large-en-v1.5 给出最佳 accuracy-latency trade-off。
  • 研究意义：为后续检索式分类、reranker serving 和 LLM inference 选型提供了更实用的参照系。
💡 核心创新点

核心思想

这篇论文的重点不在提出一个新模型，而在于把“真正 zero-shot 分类”这件事从 benchmark 层面重新定义清楚。作者把 cross-encoder、embedding、reranker、LLM 四条路线放到同一套标准下比较，并显式把 latency 与效果一起纳入考量，因此对系统和 serving 场景尤其有价值。

📊 实验结果
  • reranker 最强：Qwen3-Reranker-8B 达到新的 SOTA（macro F1 = 0.72）
  • embedding 最均衡：GTE-large-en-v1.5 在准确率和延迟之间提供最佳折中
  • LLM 逐渐可用：4B-12B 的 instruction-tuned LLM 达到最高 0.67 macro F1，尤其擅长 topic classification
  • cross-encoder 遇到瓶颈：NLI-based 路线继续扩大 backbone 收益有限

🧠 深度分析

研究价值

  • 理论价值：重新定义了 zero-shot 分类 benchmark 的评测边界，强调纯 zero-shot 而非 probe-based 评估。
  • 工程价值：对检索式分类和在线 serving 特别有意义，因为它不仅看精度，还看 latency。
  • 领域影响：可能推动更多分类系统从传统 NLI cross-encoder 转向 reranker 或 embedding route。

优势

  • benchmark 设计清晰，覆盖任务面广
  • 统一了不同模型家族的比较标准
  • 把 latency 放进主结论，对工业界更有价值

局限性

  • 这是一篇 benchmark 论文，不直接提出可复用的新 inference 方法
  • 热门度和长期影响还需要更多后续工作验证
  • 作者数较少，可能说明实验和生态建设主要靠单人推进，后续维护需观察
